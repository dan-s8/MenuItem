The goal is to cluster together restaurant brands based on their menu items being used in semantically similar contexts.


Before running the program, please install the following:
* For the purpose of training a embedding model for menu items
$ pip install nltk
$ pip install gensim


There is one jupyternotebook file: "Fengdan Shi-Food Word Embedding and Clustering Task"

You can run all cell and see the clustering result. The python scripts and functions within are commented with comments and Markdown part.
